# University of Birmingham Beamer Template

## A note on fonts

As per the University [Brand Portal](https://brand.birmingham.ac.uk/), there are two main typefaces: [Manrope](https://fonts.google.com/specimen/Manrope) (sans-serif) and [Larken](https://ellenlufftype.com/larken/) (serif). Manrope is available under the [SIL OPEN FONT LICENSE Version 1.1](https://openfontlicense.org/open-font-license-official-text/); as such, we have provided a local copy. Larken, however, is a licensed font, and cannot be provided.

The fall-back fonts are Arial (sans-serif) and Times New Roman (serif). In the author's opinion, Times New Roman is a poor choice as it does not match the modern design of the branding. As such, we have included [Marcellus](https://fonts.google.com/specimen/Marcellus?preview.script=Latn): a font closer to Larken and under the SIL OPEN FONT LICENSE Version 1.1, whilst still remaining accessible.

Should you have the licensing for Larken, you are free to upload and use that. Modify [](template/beamerfontthemeuob.sty).
